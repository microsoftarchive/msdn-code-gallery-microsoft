## Description

<div id="mainSection">
<p>This sample demonstrates the use of the XInput APIs in a C&#43;&#43; app. It reads input from an Xbox game controller and displays data about the analog stick movements and button presses.</p>
<p>Specifically, this sample covers:</p>
<ul>
<li>Getting input state from an attached controller </li><li>Drawing to the display based on controller input using Direct2D and DirectWrite
</li></ul>
<p></p>
<p>This sample is written in C&#43;&#43; and requires some experience with DirectX.</p>
<p>For more info about the concepts and APIs demonstrated in this sample, see these topics:</p>
<ul>
<li><a href="http://msdn.microsoft.com/library/windows/apps/br229580">Create an app using DirectX</a>
</li><li><a href="http://msdn.microsoft.com/library/windows/apps/ee417001">XInput</a> </li><li><a href="http://msdn.microsoft.com/library/windows/apps/dd368038">DirectWrite</a>
</li><li><a href="http://msdn.microsoft.com/library/windows/apps/dd370987">Direct2D graphics</a>
</li></ul>

<h2>Build the sample</h2>
<ol>
<li>Start Visual Studio&nbsp;2013 and select <b>File</b> &gt; <b>Open</b> &gt; <b>Project/Solution</b>.
</li><li>Go to the directory in which you unzipped the sample. Go to the directory named for the sample, and double-click the Visual Studio&nbsp;2013 Solution (.sln) file.
