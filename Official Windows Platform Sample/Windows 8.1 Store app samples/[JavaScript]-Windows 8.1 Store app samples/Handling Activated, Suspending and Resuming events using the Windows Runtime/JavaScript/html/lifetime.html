This sample app demonstrates an advanced example of activation. It shows how to register for "activated", "suspending", and "resuming" events and how to use these to save state to a file and retrieve it during re-activation. Separately, in addition to the launch activation, it also demonstrates the protocol activation as well as how to identify the various other types of activations.

To simulate suspending this sample app from within Microsoft Visual Studio Express, first launch it using Start Debugging, and then use the "Suspend" button.

To simulate resuming this sample app after suspension from within Microsoft Visual Studio Express, click on the "Resume" button.
