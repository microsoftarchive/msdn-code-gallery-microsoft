In this scenario we demonstrate how to enable the listview as a drop target. With the listview setup as an HTML 5 standard drop target (a DOM element that handles the HTML 5 drop event), the listview is ready to accept drops from HTML 5 standard sources.

This sample is much like the previous one. However, in this case we are dropping the element onto the list itself. The app computes where the item should be dropped and adds the item in the correct location (in this case, the beginning). You will note that the listview is styled on the dragOver, indicating the drop will be on the list itself, not within the list.
