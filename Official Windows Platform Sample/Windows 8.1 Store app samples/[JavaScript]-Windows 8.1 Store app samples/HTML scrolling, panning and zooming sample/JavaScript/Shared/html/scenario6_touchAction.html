When a user touches an element, that element's touch-action property determines the default touch behaviors permitted for that contact, for example panning or zooming. The following example disables a different default action for each photo and then consumes the DOM events related to the gesture in order to transform the child element.

Pinch scale

touch-action:pan-x pan-y

Drag Vertically

touch-action:pinch-zoom pan-x

Drag Horizontally

touch-action:pinch-zoom pan-y
