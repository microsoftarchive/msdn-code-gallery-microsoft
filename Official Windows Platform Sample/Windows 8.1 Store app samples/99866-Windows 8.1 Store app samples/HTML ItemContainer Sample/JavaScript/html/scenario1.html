In this scenario you'll see how to use the ItemContainer to take a block of HTML and allow it to be selected with swipe. This 'item' allows you to create selectable elements that look and feel identical to the items that are created with the WinJS listview control. In fact you'll see when looking through the MSDN documentation that you can use the familiar swipeBehavior and tapBehavior properties to customize your interactions. For example, this scenario uses the tapBehavior 'toggleSelect', to enable tap to select.
