This scenario demonstrates how to achieve an incremental loading experience with the ListView, as the 'loadingBehavior' property has been deprecated. This scenario automatically insert more data to the datasource by wrapping the itemTemplate with a function that requests/inserts more data under certain conditions. The data source used simulates a slow data source as if it was fetching data over a network.

Watch the scrollbar as more items are loaded when scrolling reaches near the end of the list.
