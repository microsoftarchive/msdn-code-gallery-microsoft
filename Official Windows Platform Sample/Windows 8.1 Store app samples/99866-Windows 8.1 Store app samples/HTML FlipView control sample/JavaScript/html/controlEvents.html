The FlipView control supports 4 events:

  • pageselected: Fires when the FlipView settles on a single item.
    After setting the currentPage, itemDataSource or itemTemplate, you should wait for this event to be invoked before triggering a navigation.
  • pagecompleted: Fires when the FlipView settles on a single item and its renderer function has completed.
  • pagevisibilitychanged: Fires when an item becomes invisible or visible.
  • datasourcecountchanged: Fires when the datasource count changes.
