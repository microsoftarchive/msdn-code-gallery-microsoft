This scenario demonstrates the simple creation of a pivot with three pivot items. Each item contains a line of text. Pivot sections can contain a wide variety of HTML / JS content, as seen in the third scenario, "Pivot with a multi-select ListView". Additionally the pivot can be launched on a given item, using the 'selectedIndex' property. This can be used for rehydration and other scenarios.
