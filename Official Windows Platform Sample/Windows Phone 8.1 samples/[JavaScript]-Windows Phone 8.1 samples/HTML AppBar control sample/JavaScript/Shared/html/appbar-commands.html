This example shows how to programmatically control the WinJS.UI.AppBar and its commands.

The Show Commands and Hide Commands buttons use the AppBar.showCommands and AppBar.hideCommands methods to control the visibility of two of the commands.

For PC, this AppBar is set to 'sticky': it is not dismissed when the user interacts with the application. If you need an AppBar to remain visible while the user interacts with content in the app (such as selecting multiple items), set the 'sticky' property to true.
