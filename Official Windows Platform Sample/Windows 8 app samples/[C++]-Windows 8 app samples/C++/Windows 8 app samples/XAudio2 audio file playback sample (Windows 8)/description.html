This sample demonstrates the use of XAudio2 in a app.

Specifically, this sample covers:

  • Parsing a RIFF audio file for the audio data
  • Setting up XAudio2 to play a sound via a buffer
  • Playing and monitoring the playstate of a sound buffer in XAudio2

This sample is written in C++ and requires some experience with graphics programming.

For more info about the concepts and APIs demonstrated in this sample, see:

  • XAudio2

Build the sample

  1. Start Visual Studio Express 2012 for Windows 8 and select File > Open > Project/Solution.
  2. Go to the directory in which you unzipped the sample. Go to the directory named for the sample, and double-click the Visual Studio Express 2012 for Windows 8 Solution (.sln) file.
  3. Press F7 or use Build > Build Solution to build the sample.

Run the sample

To debug the app and then run it, press F5 or use Debug > Start Debugging. To run the app without debugging, press Ctrl+F5 or use Debug > Start Without Debugging.
