Summary

This is a sample extension for Visual Studio 2010 that visually scales lines to be smaller the farther away from the caret's current position they are. It accomplishes this using the new line transformation API.


Requirements

  • Visual Studio 2010
  • Visual Studio 2010 SDK

Download and install

  • Download the zip file associated with the sample
  • Unzip the sample to your machine
  • Double click on the .sln file to launch the solution

Building and running the sample

  • To build and execute the sample, press F5 after the sample is loaded
