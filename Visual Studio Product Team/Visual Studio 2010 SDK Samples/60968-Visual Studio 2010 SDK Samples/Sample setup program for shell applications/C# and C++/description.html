Sample setup program for shell applications

 

Introduction:

This solution provides a sample of how to create a setup program for your Visual Studio 2012 Shell application.

An explanation to accompany this sample can be found at http://msdn.microsoft.com/en-us/library/bb932484.aspx


With this solution you can make a setup program for a Visual Studio 2012 Shell (Isolated), Visual Studio 2012 Shell (Integrated), and add any additional language packs.

Getting Started:

 

Requirements:

  • Microsoft Visual Studio 2012
  • Microsoft Visual Studio 2012 SDK
  • Microsoft Visual Studio 2012 Visualization and Modeling SDK
  • Windows Installer XML Toolset 3.6

 

Download and install:

  • Download the zip file associated with the sample
  • Unzip the sample to your machine
  • Ensure that you have all the requirements installed
  • Double click on the .sln file to launch the solution

 

Building and running the sample

  • To build the sample setup program and run the sample shell application, press F5.
  • To run the sample setup program you will need to obtain the required redistributable (VSIsoShell.exe) and place it, Setup.exe, and PhotoStudioIsoShell.msi in the same directory. Then double-click Setup.exe.
