Indexers_2 Sample

This sample shows how C# classes can declare indexers to represent an array-like collection of different kinds of things. See Properties (C# Programming Guide) for additional information.

To build and run the Indexers_2 sample within Visual Studio

  • On the Debug menu, click Start Without Debugging.

To build and run the Indexers_2 sample from the Command Line

  • Type the following at the command prompt: 

    csc indexers_2.cs
